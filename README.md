## Instructions to run

* Pre-requisites:
	- Python and pip installed
	- An active internet connection

* First we need to install the dependencies 
```bash
pip3 install -r requirements.txt
```

* To start the scraping script

```bash
export FLASK_APP=main.py
flask run
```
